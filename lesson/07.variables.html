JavaScript Variables

  • JavaScript uses the var keyword to declare variables.
  • An equal sign (=) is used to assign values to variables.

declare a variable then assign

<script> var x; // here x is declared but value is undefined document.write('value of "x" is ',x,'<br>'); x = 9; //here x's value is assigned 9 document.write('now value of "x" is ',x); </script>

declare & assign variable

<script> var x = 5; //x is defined & value is set to 5 var y = 2; //y is defined & value is set to 2 var z = x + y; //z is defined & value is set to x+y document.write('sum is',z); </script>

declaring multiple variables

<script> var p,q,r,s; p = 5; q = 3; r = 1; s = p + q - r; document.write('result is ',s); </script>

declaring & assigning multiple variables at a time

<script> var x = 5, y = 2, z = 7,a; a = z*y-z; document.write('answer is ',a); </script>
