Javascript Data Type & Structure

There are six types of data in JavaScript

জাভাস্ক্রিপ্ট এ ৬ ধরণের ডাটা আছে

  • Number (সংখ্যা)
  • String (স্ট্রিং)
  • Boolean (বুলিয়ান)
  • Object (অবজেক্ট)
    • Function (ফাংশন)
    • Array (অ্যারে)
    • Date (তারিখ)
    • RegExp (রেগুলার এক্সপ্রেশন)
  • NULL
  • undefined

How to check data type?

typeof keyword is used to check data type of a data or variable.

<script> var x,t; t = typeof x; document.write("<b>typeof</b> x is <i>",t,'</i><br>'); x = 13.8; t = typeof x; document.write("<b>typeof</b> x is <i>",t,'</i><br>'); x = 'happy coding'; t = typeof x; document.write("<b>typeof</b> x is <i>",t,'</i><br>'); x = null; t = typeof x; document.write("<b>typeof</b> x is <i>",t,'</i><br>'); </script>
