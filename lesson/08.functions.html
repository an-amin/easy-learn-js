Functions

Function is a block of code written to perform a particular task.
Function is executed when it is called.

ফাংশন তহনই কাজ করে যখন একে কল করা হয়।

Function Invocation

The code inside the function will execute when "something" invokes (calls) the function:

  • When an event occurs (when a user clicks a button)
  • When it is invoked (called) from JavaScript code
  • Automatically (self invoked)

we can set dafault value for parameters. if parameter value is not passed when the function is called, it will take default value.

Function Return

we can return a value in a function. If we don't by default function returns 'undefined'.
ফাংশনে আমরা কোনও ভ্যালু রিটার্ন করতে পারি। যদি না'ও করই তাহলে বাই ডিফল্ট ফাংশন 'undefined' ভ্যালু রিটার্ন করে।
<script> function addTwoNumbers(numb1,numb2){ return numb1 + numb2; } function addThreeNumbers(numb1,numb2,numb3){ numb1 + numb2 + numb3; } document.write(addTwoNumbers(7,5),'<br>'); //here, addTwoNumbers() returns the sum of the two parameters document.write(addThreeNumbers(7,5,9),'<br>'); //here, addTwoNumbers() returns 'undefined' </script>
when we're returning something that means the function work is done & it returns the vaule & immediately stops execution of next statements (if any).
ফাংশনে যদি কোনও ভ্যালু রিটার্ন করা হয় তাহলে ধরা হয় যে ফাংশনের কাজ শেষ, এবং ফাংশন ভ্যালু রিটার্ন করে পরবর্তী স্টেটমেন্ট (যদি থাকে) এক্সিকিউট করে না।
<script> function calculate(numb1,numb2){ return numb1 * numb2; //function is done here. so next statement will not be executed return numb1 / numb2; //this & next statements (if any) will not be executed. } document.write(calculate(15,5)); </script>
We can return something only once from a function. If we need more... we'll return them in a Object/Array.
we'll learn about Array & Object soon**
একটি ফাংশন থেকে কোন কিছু একবারই রিটার্ন করা যায়। যদি একাধিক মান/ভ্যালু রিটার্ন করতে হয় তাহলে অবজেক্ট/অ্যারে হিসেবে রিটার্ন করতে হয়।
শীঘ্রই আমরা অবজেক্ট এবং অ্যারে সম্পর্কে শিখবো**
<script> function calculate(numb1,numb2){ var obj = { addition:numb1+numb2, substraction:numb1-numb2, multiplication:numb1*numb2, divition:numb1/numb2 }; return obj; } console.log(calculate(15,5)); document.write(calculate(15,5),'<br> press F12 to open console'); </script>
We can set a function's returned value to a variable and use it anywhere.
ফাংশন থেকে প্রাপ্ত (রিটার্নড) ভ্যালু আমরা ভ্যারিয়েবলে সেট করে রেখে সেই মান ব্যবহার করতে পারি।
<script> function difference(numb1,numb2){ return numb1 - numb2; } var x = difference(10,6); document.write('difference is ',x); </script>
