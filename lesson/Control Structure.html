</script>
syntax:
(condition)?(TRUE block):(FALSE block)

if condition is TRUE, TRUE block's code will be executed,
if FALSE, FALSE block's code will be executed
examples: 

<script>
	var x = 1==1?"1 equals 1":"1 not equals 1";
	//x returns "1 equals 1" 
	
	1!=1?alert("1 not equals 1"):alert("1 equals 1");
	//pop-ups "1 equals 1"
</script>
