JavaScript Number Functions/Methods

Number is object

JavaScript number is a object and it has own property and methods.
to access it's property & methods Number.property & Number.method()

toString() method

toString() returns a number as a string

All number methods can be used on any type of numbers (literals, variables, or expressions):

<script> var x = 123; document.write('value of x is ',x,' & type of x is ',typeof x,'<br>'); var y = x.toString(); document.write('value of y is ',y,' & type of y is ',typeof y,'<br>'); </script>

toFixed() method

toFixed() returns a string, with the number written with a specified number of decimals:

<script> var x = 9.656; x.toFixed(0); // returns 10 x.toFixed(2); // returns 9.66 x.toFixed(4); // returns 9.6560 x.toFixed(6); // returns 9.656000 </script>

toFixed() method

toPrecision() returns a string, with a number written with a specified length:

<script> var x = 9.656; x.toPrecision(); // returns 9.656 x.toPrecision(2); // returns 9.7 x.toPrecision(4); // returns 9.656 x.toPrecision(6); // returns 9.65600 </script>

Converting Variables to Numbers

There are 3 JavaScript methods that can be used to convert variables to numbers. These methods are not number methods, but global JavaScript methods.

  • Number() method
  • parseInt() method
  • parseFloat() method

Number() Method

Number() can be used to convert JavaScript variables to numbers:

<script> Number(1);//returns 1 Number('1');//returns 1 Number('1 1');//returns NaN </script>

parseInt() method

parseInt() parses a string and returns a whole number. Spaces are allowed. Only the first number is returned:

<script> parseInt("3"); //returns 3 parseInt("3.25"); //returns 3 parseInt("3 2 1 0"); //returns 3 parseInt("3 idiots"); //returns 3 parseInt("the 3 idiots"); //returns NaN </script>
