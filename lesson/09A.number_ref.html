JavaScript Numbers

JavaScript has only one type of number. Numbers can be written with, or without, decimals.

<script> var x = 31.45; // A number with decimals var y = 28; // A number without decimals </script>

Extra large or extra small numbers can be written with scientific (exponent) notation:

<script> var x = 12e2; //12*10^2 var y = 123e5; //123*10^5 </script>

get type of number

<script> var x = 12; //12*10^2 var y = 123e5; //123*10^5 document.write(typeof x,'<br>'); document.write(typeof y,'<br>'); </script>

Precision

Integers are considered accurate up to 15 digits.

<script> var x = 999999999999999; // x will be 999999999999999 var y = 9999999999999999; // y will be 10000000000000000 document.write(x,'<br>'); document.write(y,'<br>'); </script>

The maximum number of decimals is 17, but floating point arithmetic is not always 100% accurate:

<script> var x = 0.2 + 0.1; // x will be 0.30000000000000004 document.write(x); </script>

To solve the problem above, it helps to multiply and divide:

<script> var x = (0.2 * 10 + 0.1 * 10) / 10; // x will be 0.3 document.write(x); </script>

Number & String operations

<script> var x = 3+5+'Apple'; document.write(x); </script>

<script> var x = 'Apple'+3+5; document.write(x); </script>

if the string contains a numeric value , the result will be a number:

<script> var p = 15/'3'; var q = 15*'3'; var r = 15+'3';//only this will be concated var s = 15-'3'; document.write(p,'<br>'); document.write(q,'<br>'); document.write(r,' concated <br>'); document.write(s,'<br>'); </script>

NaN (Not a Number)

NaN is a JavaScript reserved word indicating that a number is not a legal number.

Trying to do arithmetic with a non-numeric string will result in NaN (Not a Number):

<script> var p = 1-'car'; var q = 1+'car'; //only this will be concated var r = 1*'car'; var s = 1/'car'; document.write(p,'<br>',q,'<br>',r,'<br>',s); </script>

type of NaN is number

<script> document.write(typeof NaN); </script>

NaN operations

Mathematical operations with NaN returns NaN

<script> document.write('NaN + NaN = ',NaN+NaN,'<br>'); document.write('NaN - NaN = ',NaN-NaN,'<br>'); document.write('NaN * NaN = ',NaN*NaN,'<br>'); document.write('NaN / NaN = ',NaN/NaN,'<br>'); document.write('NaN + 123 = ',NaN+123,'<br>'); document.write('NaN - 123 = ',NaN-123,'<br>'); document.write('NaN * 123 = ',NaN*123,'<br>'); document.write('NaN / 123 = ',NaN/123,'<br>'); document.write('NaN / 0 = ',NaN/0,'<br>'); </script>

NaN operation with string

<script> var p = NaN + 'string'; //returns (string) NaNstring var q = NaN + 7 + 'string'; //returns (string) NaN7string var r = 'string' + NaN ; //returns (string) stringNaN var s = 3 + '1' + NaN ; //returns (string) 31NaN document.write(p,'<br>',q,'<br>',r,'<br>',s); </script>

Infinity

Infinity (or -Infinity) is the value JavaScript will return if you calculate a number outside the largest possible number.

<script> var x = 1/0; var y = -1/0; document.write(x,'<br>'); document.write(y,'<br>'); </script>

typeof Infinity is number

<script> document.write(typeof Infinity); </script>

Mathematical Operations with Infinity

<script> var p = Infinity + Infinity; //returns Infinity var q = Infinity - Infinity; //returns NaN var r = Infinity * Infinity; //returns Infinity var s = Infinity / Infinity; //return NaN document.write(p,'<br>',q,'<br>',r,'<br>',s); </script>

check Infinity

<script> var p = isFinite(1/0); var q = isFinite(1/1); document.write(p,'<br>'); document.write(q,'<br>'); </script>

<script> var x = new Number(7); document.write(x); </script>
