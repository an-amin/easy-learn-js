How to display something in JavaScript

  1. Writing into an HTML element, using .innerHTML
  2. Writing into the HTML document using document.write()
  3. Writing into an alert box, using window.alert()
  4. Writing into the browser console, using console.log()

Writing into an HTML element, using .innerHTML

<!DOCTYPE HTML> <html lang="en-US"> <head> <meta charset="UTF-8"> <title>.innerHTML</title> </head> <body> <h1>.innerHTML</h1> <p id="demo"></p> <p class="demo"></p> <p class="demo"></p> <p class="demo"></p> <script> // get Elements By Id document.getElementById("demo").innerHTML = "Inner HTML of Id is changed"; // get Elements By ClassName var x = document.getElementsByClassName("demo"); x[0].innerHTML = "Inner HTML of First Demo Changed "; x[1].innerHTML = "Inner HTML of Second Demo Changed "; </script> </body> </html>

Writing into the HTML document using document.write()

<!DOCTYPE HTML> <html lang="en-US"> <head> <meta charset="UTF-8"> <title>document.write()</title> </head> <body> <h2>document.write()</h2> <p></p> <script> document.write('Hello world. <br> '); document.write('Writing with document.write() <br> '); </script> </body> </html>

Writing into an alert box, using window.alert()

<!DOCTYPE HTML> <html lang="en-US"> <head> <meta charset="UTF-8"> <title>window.alert()</title> </head> <body> <h2>window.alert()</h2> <script> window.alert("Don't worry, this is a simple alert!"); </script> </body> </html>

Writing into the browser console, using console.log()

<!DOCTYPE HTML> <html lang="en-US"> <head> <meta charset="UTF-8"> <title>console.log()</title> </head> <body> <h2>console.log()</h2> <p>Press <b>F12</b> to Open Browser Console</p> <script> console.log("Hello from Browser Console."); console.log("Welcome once again."); console.log("please type '2+3' & press ENTER."); </script> </body> </html>
