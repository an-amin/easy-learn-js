How to write JavaScript?

in HTML document

  1. to write JavaScript in HTML we must use <script> tags. Code goes in between <script> </script> tags.
    <script> document.write("Hello from JavaScript."); //other codes; </script>
  2. <script> tags can be placed anywhere in HTML document, but usually it is written in <head> and <body>
    <!DOCTYPE HTML> <html lang="en-US"> <head> <meta charset="UTF-8"> <title>JavaScript</title> <script> //JavaScript Codes; </script> </head> <body> <h1>Title</h1> <p>paragraph</p> <script> //JavaScript Codes; </script> </body> </html>
  3. the best practice to place <script> tags in HTML is to place it as lower as possible in the <body> tags, so that all the elements the document are surely loaded & JavaScript can access them & runs the application flawlessly.

Inline with HTML tag

  1. we can also write inline JavaScript in HTML event attributes like: onclick, onmouseover, onmouseout, onload etc....
    <!DOCTYPE HTML> <html lang="en-US"> <head> <meta charset="UTF-8"> <title></title> </head> <body> <button onclick="this.innerHTML=Date();">click to show date & time</button> <br> [Date & Time will be shown on THIS element] <br><br> [Play with Mouse] <br> <button onclick="this.innerHTML='Clicked';" onmouseover="this.innerHTML='Mouse In';" onmouseout="this.innerHTML='Mouse Out';" > Move Mouse Around Here </button> </body> </html>

in external file

  1. we can write JavaScript in an external file, and link it into HTML in <head> and <body> tags wherever need.
  2. JavaScript file format is .js and we don't need to write <script> tag in this file.
    <script src="file_name.js"> </script>
    we cannot write anything between this <script> tags. If we need any we'll write new <script> tags.
  3. the best practice to link an extarnal .js file is lower as possible in the <body> tag for the very same reason described earlier.
  4. follow the example bellow:
    HTML file index.html
    <!DOCTYPE HTML> <html lang="en-US"> <head> <meta charset="UTF-8"> <title>External JavaScript</title> </head> <body> <h1>External JavaScript</h1> <script src="code.js"></script> </body> </html>

    JavaScript file code.js
    document.write("Hello from JavaScript."); //other codes;
