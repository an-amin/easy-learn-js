Conditions in JavaScript

conditional operators

Operator Name & Role Example

  ==


double equal sign 		if is equal
checks if the values is equal 		1==1;  //returns TRUE
1=='1';  //returns TRUE
1==2;  //returns FALSE

 ===


triple equal sign 		if strickty is equal
checks if both the values & types are equal 		1===1;  //returns TRUE
1==='1';  //returns FALSE
1==2;  //returns FALSE

  !=


exclumation & equal sign 		if is not equal
checks if the values is not equal 		1!=1;  //returns FALSE
1!='1';  //returns FALSE
1!=2;  //returns TRUE

  !==


exclumation & double equal sign 		if strictly is not equal
checks if both the values & types are not equal 		1!==1;  //returns FALSE
1!=='1';  //returns TRUE
1!==2;  //returns TRUE

  &&


double ampersand sign 		Logical AND
join two or more conditions,
and checks iff each are TRUE 		(condision1)&&(condition2);
in case, all conditions are TRUE, returns TRUE,
in all other cases returns FALSE
example:
(TRUE)&&(TRUE);  //returns TRUE
(TRUE)&&(FALSE);  //returns FALSE
(FALSE)&&(TRUE);  //returns FALSE
(FALSE)&&(FALSE);  //returns FALSE
(TRUE)&&(TRUE)&&(FALSE);  //returns FALSE

  ||


double pipe sign 		Logical OR
join two or more conditions,
and checks if/atleast any any of them are TRUE 		(condision1)||(condition2);
in case, all conditions are FALSE,
returns FALSE,
in all other cases returns TRUE
example:
(FALSE)||(FALSE);  //returns FALSE
(TRUE)||(FALSE);  //returns TRUE
(FALSE)||(TRUE);  //returns TRUE
(FALSE)||(FALSE);  //returns TRUE
(FALSE)||(TRUE)||(FALSE);  //returns TRUE
